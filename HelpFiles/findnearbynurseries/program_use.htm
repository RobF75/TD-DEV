Using the Program

Type a 4 digit postcode into the datafield labelled as 'Postcode' located in the application toolbar. This is used as the point of origin for the search.

Specify a search distance (radius) to scan by entering any numeric value up to 99 into the datafield labelled 'Search Distance (kms)'.
If no distance is specified, the seach is limited to using only the specified postcode.

Click on the Populate button to start the search. This may take a minute or two to complete.

A progress bar will show the search status.

Once the results are displayed, they may be printed or saved to a file as required by selecting the appropriate toolbar button.
Individual resultant rows can also be selected and copied by right clicking on the results table and choosing the copy option.
