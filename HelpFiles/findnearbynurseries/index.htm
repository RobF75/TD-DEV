Program Use

Find Nearby Nurseries

This program is designed to provide a list of Garden Centres or Nuseries that have purchased Fleming's products.

It can be used to provide customers with points of sale of Fleming's stock in their local area.

The results are determined using the following method:

  • The supplied postcode is used to provide a central point of origin.
  • If a distance is supplied, it is used for specifying the search radius around the given postcode.
  • Any other postcodes that fall within the range of the specified distance are collected into a list.

Details of any Garden Centres with a 'Main' business address that includes a postcode contained in the list are obtained.

  • Only Garden Centres that are the Fleming's mailing list are shown.
  • If the Current Season Only box is ticked, only Garden Centres who have placed current season's orders are shown.

The data that is shown is initially listed in order of distance from the point of origin.
All columns in the results table are able to be used for sorting. Simply double-click on a column header to sort the results into ascending or descending order based on that column.

Further detailed instructions are found in the topic Using the Program
